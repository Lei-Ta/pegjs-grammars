h1. eJSON

Grammar file of Enhansed JSON for PEG parser generator

The grammar adds QML-like features to JSON objects: 

* Keyless fields
* Named objects
* Alternatives




{ 
  "doctype" { "html" },
  "html" {
    "head" { 
      "title" { "This is title"},
      "meta"  { "charset" : "utf-8"} 
    },
    "body" {
      "h1" { "id" : "title", "This is header" },
      "p"  { "This is paragraph" },
      "ul" { "li" [
         {"One"},
         {"Two"},
         {"Three"}
      ]} 
    } 
  }
}

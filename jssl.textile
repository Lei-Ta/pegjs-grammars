h1. JSSL

"Grammar file":https://github.com/Re-Ling/pegjs-grammars/blob/master/grammars/JSSL.pegjs of JS Serialization Language for PEG parser generator

The grammar adds QML-like features to JSON objects: 

* Keyless values
* Named objects
* Duplicated objects 

Addiyionally, JSSL has repetitions.

JSSL can be used as replacement to XML and HTML, because it allows objects with same name and preserves order, and its syntax is simpler then JSON's one.

Below there is code in JSSL and parser's output in JSON:

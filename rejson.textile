h1. reJSON

"Grammar file":https://github.com/Re-Ling/pegjs-grammars/blob/master/grammars/reJSON.pegjs of Enhansed JSON for PEG parser generator

The grammar adds QML-like features to JSON objects: 

* Keyless values
* Named objects
* Repetitions

reJSON can be used as replacement to XML and HTML, because it allows objects with same name and preserves order, and its syntax is simpler then JSON's one.

Below there is code in reJSON and parser's output in JSON:

h1. ReML

Grammar file of ReML for PEG parser generator

ReML can be used as replacement to XML and HTML, because it allows objects with same name and preserves order, and its syntax is simpler then JSON's one.

There is code in ReML and parser's output in JSON: 

Input (ReML):

<pre><code>(
  doctype 'html'
  html ( 
    head ( 
      title 'This is title'
      meta ( charset = 'utf-8' ) 
    ) 
    body ( 
      h1#phead 'This is header'
      p 'This is paragraph' 
      ul li (
        'One' 
      | 2
      | 'Three'
      )
    ) 
  )
)</code></pre>

Ouput (JSON):
